Local Devving:

To develop:

```
  npm run dev:docker
  npm run dev:seed
  npm run dev:turbo
```

To run a production version:

```
  npm run dev:docker
  npm run dev:seed
  npm run dev:build
  npm run dev:deploy
```

Bonus!:
To run the e2es, after performing the prod version steps, do `npm run test:e2e`
